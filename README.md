# Sistema de Gestão de Farmácia

> Sistema web desenvolvido em Python para apoiar a gestão e organização das operações de uma farmácia.

## Sobre o projeto

O **Sistema de Gestão de Farmácia** é uma aplicação web desenvolvida com **Python e Flask**, criada para digitalizar e organizar processos relacionados à gestão de uma farmácia.

O projeto foi desenvolvido com foco na **gestão de produtos, organização de dados e disponibilização de informações através de uma interface web**.

Este projeto representa uma aplicação prática de conhecimentos em **desenvolvimento web, programação em Python, bases de dados e organização de sistemas**.

---

## Objetivos

* Desenvolver uma solução web para apoiar a gestão de uma farmácia;
* Organizar informações relacionadas aos produtos;
* Facilitar o acesso e a gestão dos dados do sistema;
* Aplicar conceitos de desenvolvimento web com Python;
* Utilizar uma base de dados para persistência das informações;
* Desenvolver uma aplicação com estrutura modular.

## Como executar o projeto

### 1. Clonar o repositório

```bash
git clone git@github.com:Estanislau-Eng/SIstema_Farmacia.git
```

### 2. Aceder à pasta do projeto

```bash
cd SIstema_Farmacia
```

### 3. Criar um ambiente virtual

```bash
python -m venv venv
```

### 4. Ativar o ambiente virtual

#### Linux/macOS

```bash
source venv/bin/activate
```

#### Windows

```bash
venv\Scripts\activate
```

### 5. Instalar as dependências

```bash
pip install -r requirements.txt
```

### 6. Executar a aplicação

```bash
python app.py
```

Depois, aceda à aplicação através do endereço indicado pelo Flask no terminal.
